cudnn
==============================

Description
~~~~~~~~
cuDNN is a deep neural network library from Nvidia that provides a highly tuned implementation of many functions commonly used in deep machine learning applications.

Versions
~~~~~~~~
- cuda-11.0_8.0
- cuda-11.2_8.1
- cuda-11.4_8.2
- cuda-12.0_8.8

Module
~~~~~~~~
You can load the modules by::

    module load cudnn
