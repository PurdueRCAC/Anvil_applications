gcc
==============================

Description
~~~~~~~~
The GNU Compiler Collection includes front ends for C, C++, Objective-C, Fortran, Ada, and Go, as well as libraries for these languages.

Versions
~~~~~~~~
- 8.4.1
- 11.2.0
- 11.2.0-openacc

Module
~~~~~~~~
You can load the modules by::

    module load gcc
